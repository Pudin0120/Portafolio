# Online Book Store

## Description
Project Online Book Store: management system built with Java, HTML/CSS, JSP/Servlets, XML. This folder contains the source code and resources needed to review, run, and adapt the solution as part of the portfolio.

## What It Does
- Provides a practical solution related to the project name: Online Book Store.
- Includes source files, resources, and configuration needed to study or run the application.
- Works as an individual project entry in Joseff Antonio Laverde Avendano's portfolio.

## Detected Technologies
- Java
- HTML/CSS
- JSP/Servlets
- XML

## Requirements
- JDK 8 or newer
- A Java server compatible with Servlets/JSP, such as Apache Tomcat

## Installation
1. Clone or download this portfolio repository.
2. Open the Projects/Online Book Store folder.
3. Open the project in the matching editor or IDE, then compile or run the main file.

## Usage
- `javac "order/Client.java"`
- `java Client`

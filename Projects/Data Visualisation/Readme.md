## Description
Project Data Visualisation: application built with XML. This folder contains the source code and resources needed to review, run, and adapt the solution as part of the portfolio.

## What It Does
- Provides a practical solution related to the project name: Data Visualisation.
- Includes source files, resources, and configuration needed to study or run the application.
- Works as an individual project entry in Joseff Antonio Laverde Avendano's portfolio.

## Requirements
- A code editor and the runtime required by the source files

## Installation
1. Clone or download this portfolio repository.
2. Open the Projects/Data Visualisation folder.
3. Download dependencies and build with `mvn clean install`.

## Usage
- `mvn clean install`
- `mvn exec:java  # if the pom defines a main class`

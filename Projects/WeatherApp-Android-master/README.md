# WeatherApp-Android-master

## Description
Project WeatherApp-Android-master: automation utility built with Java, HTML/CSS, XML. This folder contains the source code and resources needed to review, run, and adapt the solution as part of the portfolio.

## What It Does
- Provides a practical solution related to the project name: WeatherApp-Android-master.
- Includes source files, resources, and configuration needed to study or run the application.
- Works as an individual project entry in Joseff Antonio Laverde Avendano's portfolio.

## Detected Technologies
- Java
- HTML/CSS
- XML

## Requirements
- JDK 8 or newer

## Installation
1. Clone or download this portfolio repository.
2. Open the Projects/WeatherApp-Android-master folder.
3. Build with Gradle using `./gradlew build` or `gradlew.bat build` on Windows.

## Usage
- `.\gradlew.bat build`
- `.\gradlew.bat run  # if the project defines a run task`
- `Open in a browser: index.html`

# E-commerce-project-springBoot-master

## Description
Project E-commerce-project-springBoot-master: application built with Java, JSP/Servlets, XML, SQL. This folder contains the source code and resources needed to review, run, and adapt the solution as part of the portfolio.

## What It Does
- Provides a practical solution related to the project name: E-commerce-project-springBoot-master.
- Includes source files, resources, and configuration needed to study or run the application.
- Works as an individual project entry in Joseff Antonio Laverde Avendano's portfolio.

## Detected Technologies
- Java
- JSP/Servlets
- XML
- SQL

## Requirements
- JDK 8 or newer
- A Java server compatible with Servlets/JSP, such as Apache Tomcat
- A database engine compatible with the included SQL scripts

## Installation
1. Clone or download this portfolio repository.
2. Open the Projects/E-commerce-project-springBoot-master folder.
3. Download dependencies and build with `mvn clean install`.

## Usage
- `mvn clean install`
- `mvn exec:java  # if the pom defines a main class`

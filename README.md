# Project Portfolio

General project portfolio by Joseff Antonio Laverde Avendaño. This repository groups applications, utilities, games, algorithm exercises, management systems, web projects, backend services, Android projects, and academic work in one place.

## Author
Joseff Antonio Laverde Avendaño

## How To Use This Portfolio
1. Review the project list on this page.
2. Open the project folder you want to run inside `Projects/`.
3. Read that project's `README.md` for requirements, installation steps, and usage commands.
4. Run the project with the matching runtime: Java/JDK, Python, Node.js, a web browser, Maven, Gradle, Tomcat, Android Studio, or another required tool.

## General Requirements
- Git for cloning or versioning the portfolio.
- JDK for Java projects.
- Python 3 for Python projects.
- Node.js for JavaScript, TypeScript, or React projects.
- Maven or Gradle when the project includes `pom.xml`, `build.gradle`, or a Gradle wrapper.
- An IDE such as Visual Studio Code, IntelliJ IDEA, NetBeans, Eclipse, or Android Studio depending on the project type.
